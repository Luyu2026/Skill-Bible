# Transcript Answer Rubric

用于判断面试录音复盘文档是否真的可用。

## 必须达标

| 指标 | 达标标准 |
|-|-|
| 问题还原 | 能从转写稿中提炼真实问题和追问，不遗漏主要压力点 |
| 原回答诊断 | 指出具体问题，例如太理论、无数据、没答到点、暴露风险 |
| 下次可用 | P0/P1 题都有可直接开口答案 |
| 数据密度 | 专业题答案包含指标、阈值、动作顺序或业务口径 |
| 风险控制 | HR 题不抱怨前公司，不扩大个人短板 |
| 岗位贴合 | 答案能回到岗位职责和面试官关注点 |
| 练习价值 | 输出补充材料和短期练习计划 |

## 常见失败

- 只做会议纪要式总结，没有生成下次答案。
- 把录音里的原回答润色一下，但没有补足缺失的业务动作。
- 答案仍然像方法论，没有具体数字和操作顺序。
- HR 回答过度真诚，暴露稳定性或情绪风险。
- 输出“替换字段”，但用户已经提供了自己的经历。
