# 风格指南

来源：12篇高传播视频文案的结构化拆解

## 核心三层结构

```
第一层：情绪冲击
  → 第一句话就捅刀子
  → 让读者觉得你在说他
  → 不说"你有没有想过"，说"你想过没有。没有。我告诉你"

第二层：信息量
  → 给一个他没想到的角度
  → 用场景、数字、对比讲透
  → 自然引入一次名人观点
  → 不用"他曾说过"，用故事带观点

第三层：抓手
  → 给一个能带走的东西
  → 清单 / 三步 / 三个问题 / 算账模板
  → 具体到明天就能用
```

## 节奏规则

### 1. 不要均匀短句

不要：
```
你看到同事升职
你觉得他靠关系
有没有可能
他比你更拼
```

要：
```
你看到同事升职了（缓一缓，让人想一下）
觉得他是靠关系（扎一刀）
有没有一种可能（顿一下）
他每天最后一个走（再扎一刀）
你根本没看见（最后一刀）
```

一句喘气。一句扎心。错落才有力。

### 2. 人称

- 全程"你"。一对一对话感。
- 不用"我们"、"大家"。
- 直接说"我问你"、"我告诉你"。

### 3. 反问制造停顿

- 你知道为什么吗？
- 你以为这就完了？
- 你有没有想过另一种可能？

## 引用规则

1. **每次只引一个人**
2. **优先级：纳瓦尔 → 芒格 → 段永平**
3. **不提第二句**。一句带过。
4. **引用必须真实**。这个人确实表达过类似观点。不要求逐字对照，自然引述即可。不确定时宁可不引。
5. **用故事代替死板引用**
   - 正确："纳瓦尔说过，你所有的痛苦都来自拒绝接受世界本来的样子"（自然，像聊天）
   - 错误："纳瓦尔在《纳瓦尔宝典》第132页提到…"（太正式，像论文）
6. **引不到就不引**

## 绝对避免

输出前核对。踩到就改。

1. **"不是A，而是B"滥用**。整篇最多用一次。
2. **开头下定义**。"XXX是指…"。不要解释概念。
3. **结尾喊口号**。"让我们一起…"。金句收尾，不是鸡汤。
4. **空洞排比**。三个排比没有信息增量就是废话。
5. **中立废话**。"凡事都有两面性"。等于什么都没说。
6. **双引号刷屏**。好文案不需要画重点。
7. **"首先、其次、最后"**。结构应该自然流动。
8. **"众所周知"、"不言而喻"**。读者不知道。

## 替代表达

| 不要用 | 换成 |
|--------|------|
| 我们要学会… | 你得… |
| 这件事很重要 | 这件事能要你的命 |
| 大家都有不同看法 | 你俩看的是同一件事，结论相反 |
| 我们要努力 | 你每天加班到几点，工资涨了吗 |
| 从另一个角度来看 | 有没有一种可能 |
| 正如XX所说 | 纳瓦尔在《纳瓦尔宝典》里提过一个观点 |
| 总而言之 | （直接说结论，不需要总结） |

## 钩子开场

- 反常识断言："赚钱的第一性原理就是不干活"
- 颠覆认知："真正的有钱人都不搭理任何人"
- 直击痛点："乱花钱的都是傻子"
- 设问开场："什么样的人能在股市赚钱？"

## 输出要求

每篇输出必须包含：
1. 一个让人看完想转发的金句
2. 一件能带走的东西（清单/工具/问题）
3. 两个版本（原版+剪辑版）
4. 字数统计附在末尾

## 长度

- 500-1500字
- 适合1-3分钟口播
